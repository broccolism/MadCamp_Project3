- 당근 1개를 먹으면 HEALTH && HUNGER 가 모두 FULL 상태가 돼요
- HEALTH: 물에 들어가면 아파요
- HUNGER: 버섯을 안 먹으면 배고파요
- STAMINA: 여러번 점프하면 힘들어요
- HEALTH나 HUNGER가 0이 되면 세상을 떠나고 말아요😥
- 여우를 만나면 절 더이상 볼 수 없어요😥
